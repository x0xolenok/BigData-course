# Methods and information technologies of Big Data processing 
Course at V.N Karazin Kharkiv National University
# Course structure:
1. Introduction to the course - Basic concepts, tools and issues
2. MapReduce - Overview of the method, advantages and disadvantages
3. The necessary basics of Matrix Algebra
- Basic concepts
- Projection and isometry
4. Multivariate distributions and moments
- Random vectors
- Mathematical expectation and covariance
- Conditional distribution
- Maximum likelihood estimation
5. Dimensionality reduction
- Principal Component Analysis (PCA)
- Multidimensional Scaling
- Diffusion Maps
6. Classification and clustering - Discriminant and Cluster Analysis
7. SVM - Basic concepts
8. Machine learning - Learning with and without a supervisor
